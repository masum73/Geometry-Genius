What is DOM and what is the purpose of DOM?

DOM means Document Object Meaning. In javaScript, we can change the html document structure, style, and content.

How will you select HTML elements using DOM? Name the DOM methods.

By using the below DOM methods we can select HTML elements. For example: if we want to use getElementById() method then we must declare an id to the html element that we want to select.

DOM Methods:

1.getElementsByTagName()

2.getElementsByClassName()

3.getElementById()

4.querySelector()

5.querySelectorAll()

What is event bubble?

Event Bubbling happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element. On other words, bubbling is basically traveling from child element to root parent element.

What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function so that it can be executed in that other function.
